REFACTOR PROJECT
pivarshev.github.io
================================================================

This folder contains documentation for the planned code
refactor of Pavel Pivarshev's portfolio site.

No visual or UI changes are proposed. All work targets
code quality, maintainability, and best practices.


FILES IN THIS FOLDER
----------------------------------------------------------------

site_audit.txt
  Full audit of the codebase covering 11 issues ranked by
  severity, with code examples and explanations. This is
  the primary reference document.

REFACTOR_REPORT.md
  Detailed markdown report with the same findings in a
  format suitable for GitHub viewing. Includes tables and
  formatted code blocks.

phase1_checklist.txt
  Step-by-step checklist for Phase 1 (quick wins). These
  are low-risk changes that can be made without altering
  page structure.

phase2_checklist.txt
  Step-by-step checklist for Phase 2 (structural refactor).
  These changes restructure the HTML but keep visual output
  identical.

phase3_checklist.txt
  Step-by-step checklist for Phase 3 (enhancements).
  Optional improvements for performance and developer
  experience.


HOW TO USE
----------------------------------------------------------------

1. Read site_audit.txt for the full picture
2. Work through phases in order (1 → 2 → 3)
3. Each phase has its own checklist file
4. Check off items as they are completed
5. Test each change in browser before committing
   (desktop and mobile views)
